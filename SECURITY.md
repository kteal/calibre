# Security

Report a path-containment flaw, data-loss bug, or library-corruption risk
through GitHub's private security advisory form for this repository. Do not
attach a real library. Build a minimal temporary reproduction.

The supported security boundary appears in `docs/compatibility.md`. Concurrent
writes from Calibre and this crate remain unsupported.

The maintainers will acknowledge a report when they can reproduce it. No fixed
response or release schedule applies while the project remains pre-1.0.
